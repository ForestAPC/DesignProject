To run this code

1. Download the zip file
2. Open it to a new source file of the project you want to edit. 
3. Take all of the images, text files, and sounds out of the source folder and into the main sub folder
	Ex. User/eclipse-workspace/ProjectFilename/ instead of 					User/eclipse-workspace/ProjectFilename/src/package/

4. In eclipse, run the class DrawingFrame
5. Play the game
